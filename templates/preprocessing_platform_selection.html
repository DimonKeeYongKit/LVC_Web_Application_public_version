Preprocessing Data


This preprocessing data module will be executed first before progress to the analysis part. This module is mainly responsible for processing data and sorting the data to meet the input conditions of the analyzer. At the same time, this module will remove some unnecessary or useless content for the analysis results, such as web links, hashtags, and so on. This can avoid affecting the analysis results of the analyzer of the system and reduce the burden of the analyzer for analyzing large amounts of data.
