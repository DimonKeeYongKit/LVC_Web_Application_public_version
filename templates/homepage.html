Introduction


Light Verb Construction(LVC) web application is an online web application that mainly focuses on providing the light verb construction extraction functionality. This web application helps linguists to simplify the corpus data collection and set up a platform for linguists to share data and information. The biggest feature of this system is that it has built-in algorithms to automatically recognize Malaysian English Light Verbs, which is one of the famous topics for linguists to make analysis and research in Malaysia. This web application also has a great contribution to the field of artificial intelligence, especially natural language processing, which includes algorithms and techniques used and executed. This is also very helpful for Malaysia's future technological development in these related professional fields.
